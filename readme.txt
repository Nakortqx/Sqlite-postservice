 
 Для запуска выполнить скрипт main.py,
 Сервер находится на стандартном адресе 127.0.0.1:8000

 Доступные адреса:
 /requests
 /post_offices
 /clients
 /employees

 Доступно добавление почтовых офисов, клиентов и сотрудников, а так же просмотр,
 редактирование и удаление по id; просмотр результатов некоторых запросов к бд
 по адресу /requests.

 Бд заполняется исходными данными при запуске и очищается при выключении.
